# Sentry's Open Source Pledge

We started the [Open Source Pledge](https://opensourcepledge.com/) in 2023.
This repo is where we manage [our own annual Pledge
commitments](https://opensourcepledge.com/members/sentry/). Here's the [current
issue](https://github.com/getsentry/open-source-pledge/issues/249).

Summary (details in reports linked from Pledge
[page](https://opensourcepledge.com/members/sentry/)):

- Bulk of budget goes through [Thanks.dev](https://thanks.dev/), work with Ali
  on it. He manages both foundation sponsorships (email back-and-forth) and
long-tail distribution throught the platform. Pretty smooth at this point but
does require some oversight. Includes soliciting nominations from Sentry
internal (via a Google Form) that Ali then manually includes.

- 10% of budget (including their fee) through Open Source Collective's
  [funds.ecosyste.ms](https://funds.ecosyste.ms/). Just rinse-and-repeat from
last year.

- Bare minimum ($50k? I think is the invoice billing threshold) through [GitHub
  Sponsors](https://github.com/open-source/sponsors). This goes on the
[Syntax.fm org account](https://github.com/syntaxfm) for them to distribute as
they see fit.
